# Indian Rocks Beach Guide

A single-page trip companion for visitors to Indian Rocks Beach, Florida — covering beach conditions, rules, local attractions, and dining recommendations.

**Live site:** https://[username].github.io/indian-rocks-beach/

## Sections

1. **Hero** — Introduction and orientation
2. **Beach Conditions & Essentials** — Water temps, parking, amenities, best times
3. **Beach Rules & Tips** — Official rules and local pro tips
4. **Local Attractions** — Six curated attractions and activities
5. **Dining & Drinks** — Five local restaurant recommendations

## Tech

Static HTML/CSS/JS — no framework, no build step, no external API dependencies. Deployed via GitHub Pages.

## Design

- Fonts: [Fraunces](https://fonts.google.com/specimen/Fraunces) (display) + [Atkinson Hyperlegible](https://fonts.google.com/specimen/Atkinson+Hyperlegible) (body)
- Color palette: Gulf deep navy, seafoam teal, warm sand, coral accent
- Mobile-first, responsive at 375px / 640px / 1024px / 1280px
- WCAG AA contrast compliant
